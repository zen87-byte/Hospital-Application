Untuk meng-compile aplikasi, diperlukan visual studio yang sudah di-install dengan fitur Development with C/C++

1. Unduh visual studio di sini
https://visualstudio.microsoft.com/downloads/

2. Install visual studio dengan fitur Development with C/C++

3. Buka folder hospital-management di x64 Native Tools Command Prompt for Visual Studio 2022

4. Jalankan compile.bat melalui command prompt tersebut

5. Tunggu hingga instalasi selesai

6. Akan muncul file executable bernama hospital-management.exe

7. Jalankan hospital-management.exe
